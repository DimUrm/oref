# Oref Alert для MajorDoMo (v1.44 Final)

Интеграция системы домашней автоматизации **MajorDoMo** с серверами Службы тыла Израиля (**Pikud HaOref / Командование тыла**). Модуль обеспечивает мониторинг сигналов тревоги в реальном времени, визуализацию на карте и автоматизацию защитных сценариев.

## 🚀 Основные возможности

*   **Адаптивный мониторинг (2/8):** Опрос API каждые 2 секунды во время активности и 8 секунд в режиме покоя.
*   **Двойной контроль:** Одновременная проверка Real-time API и History API для исключения пропусков при сетевых сбоях.
*   **Интеллектуальная Kiosk-карта:**
    *   Отрисовка полигонов зон обстрела по всей стране.
    *   Автоматический фокус (зум) на активных зонах.
    *   Плавный возврат к общему виду Израиля при отбое.
    *   **Burn-in protection:** Плавающие часы и элементы интерфейса для защиты OLED-экранов от выгорания.
    *   **Blur-эффект:** Размытие карты в режиме тишины для эстетичного вида.
*   **Зонирование:** Мониторинг как всей страны, так и конкретных «Моих зон» (выделяются приоритетным цветом).
*   **Гео-локация:** Расчет точного расстояния от вашего дома до эпицентра тревоги.
*   **База данных:** Встроенный справочник (1450+ городов) с временем до укрытия на русском, иврите и английском.
*   **Кастомные сценарии:** Выполнение PHP-кода при тревоге (закрытие жалюзи, включение света, оповещение в Telegram).

## 🛠 Установка

1.  Скачайте архив модуля.
2.  Распакуйте содержимое в корневую папку вашего MajorDoMo.
3.  Пути файлов должны соответствовать структуре:
    *   `/modules/oref_alert/`
    *   `/templates/oref_alert/`
    *   `/scripts/cycle_oref_alert.php`
    *   `/cms/oref_alert/`
4.  Перейдите в панель управления MajorDoMo -> **Система** -> **Проверка обновлений** -> **Маркет** (или установите через "Добавить вручную").
5.  После установки модуль автоматически создаст класс `Warning` и объект `Alarm`.

## ⚙️ Настройка

### 🎯 Мои зоны
Впишите названия районов на иврите через запятую (например, `קריית ביאליק, חיפה`). Модуль будет выделять их в истории и использовать для активации Модального окна на карте.

### 📍 GPS Координаты
Укажите широту и долготу вашего дома. Это необходимо для работы функции «Дистанция до угрозы».

### 🔔 Сценарии (Actions)
На вкладке **Действия** можно прописать PHP-код:
*   **При тревоге:** `say("Воздушная тревога!", 2);`
*   **При отбое:** `say("Отбой тревоги", 2);`

## 📺 Монитор (Kiosk Mode)

Полноэкранная карта для планшетов на стене доступна по адресу:
`http://[IP_ВАШЕГО_СЕРВЕРА]/cms/oref_alert/oref_map.php`

**Особенности монитора:**
*   **Красный экран:** Если тревога в ваших зонах.
*   **Оранжевые полигоны:** Если тревога в других частях страны.
*   **Боковая панель:** Группировка всех городов по типам угроз (Ракеты, БПЛА и т.д.) за последние 60 секунд.
*   **Zero Flicker:** Обновление списка городов происходит без мерцания интерфейса.

## 🎛 Объектная модель

Модуль наполняет свойства объекта `Alarm` (или того, который вы указали):

| Свойство | Описание |
| :--- | :--- |
| `.Status` | Текущий статус (`Alert` / `No Alert`) |
| `.City` | Список всех активных городов в стране |
| `.MyActiveAreas` | Список ваших городов под обстрелом |
| `.Distance` | Расстояние в км до ближайшей активной зоны |
| `.Category` | ID категории (1 - Ракеты, 6 - БПЛА, 10 - Отбой и т.д.) |
| `.History` | Бегущая строка (уникальные города за час по всей стране) |
| `.ShelterTime` | Время в секундах до укрытия для вашей зоны |
| `.HistoryData` | JSON-массив последних 30 событий (для карты и логов) |

## ⚠️ Важное примечание
Если ваш сервер находится **вне Израиля**, API Службы тыла может блокировать запросы. В этом случае в настройках модуля необходимо указать рабочий израильский HTTP-прокси.
