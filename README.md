# 🚨 OrefAlert для MajorDoMo

Модуль мониторинга тревог гражданской обороны Израиля (Пикуд ха-Орэф).

## Возможности

- **3 источника данных** (real-time + 2 history API) — надёжнее одного
- **1450 городов Израиля** с точным временем укрытия (из cities.json)
- **Полигоны районов** на интерактивной карте Leaflet (из polygons.json)
- **28 категорий тревог**: ракеты, БПЛА, землетрясение, цунами, химия...
- **Учения игнорируются** (категории 15+)
- **Логика 2/8 сек**: при изменении данных — 2 сек, иначе — настроенный интервал
- **Автовсплытие карты** при тревоге (настенный терминал / отдельное окно)
- **Совместимость** с существующим объектом Alert и виджетом Warning
- **Кастомный PHP-код** для каждого типа события (вкладка Действия)
- **Русские названия** городов и инструкций, настраиваемые через UI

## Установка

1. Установить через Маркет MajorDoMo или загрузить архив вручную
2. При установке автоматически создаётся объект **Alert** со всеми свойствами
3. Открыть **Система → Циклы → cycle_oref_alert → Запустить**
4. Открыть модуль **OrefAlert**, вкладка **Настройки**
5. Указать свои города (иврит, поиск работает на русском: "Кирьят")
6. Настроить метод Trigger объекта Alert под свою систему

## Свойства объекта Alert

| Свойство | Описание |
|---|---|
| `Alert.Status` | `Alert` / `No Alert` |
| `Alert.state` | `ok` / `pre_alert` / `alert` (HA-стиль) |
| `Alert.Name` | Русское название тревоги |
| `Alert.Instructions` | Русская инструкция |
| `Alert.Img` | Путь к иконке категории |
| `Alert.City` | Район на иврите |
| `Alert.CityRu` | Район на русском |
| `Alert.ZoneRu` | Округ на русском |
| `Alert.CityID` | ID города (для карты) |
| `Alert.MapData` | JSON с полигоном для Leaflet |
| `Alert.ShelterTime` | Секунд до укрытия |
| `Alert.Countdown` | Обратный отсчёт (рассчитывается в шаблоне) |
| `Alert.History` | Города за последний час |
| `Alert.Count` | Счётчик тревог |

## Карта (map_popup.html)

Откройте в браузере: `http://[ip]/templates/oref_alert/map_popup.html`

- **Режим ожидания**: часы, дата, флаг, «Всё спокойно»
- **Тревога**: автополноэкранный режим, красный полигон района, обратный отсчёт
- **Отбой**: зелёный экран 8 сек → возврат в режим ожидания

## Автовсплытие на любой странице

```html
<script src="/templates/oref_alert/alert_watcher.js"></script>
```
